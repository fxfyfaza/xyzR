## Installation

Install my-project with git-clone

```bash
  git clone https://github.com/fxfyfaza/xyzR
```

Masuk kedalam Directory File RAFT

```bash
  cd xyzR/
```

Jalankan Tools menggunakan command `python3`
```bash
  python3 xyzR.py

```
